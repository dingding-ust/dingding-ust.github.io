# Personal Website

This is a no-build static personal homepage. You can open it directly in a browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Edit your content

Update [`site.config.js`](./site.config.js) to replace the starter content:

- `profile`: name, affiliation, email, GitHub, Scholar, LinkedIn, CV
- `about`: short bio paragraphs
- `research`: three concise themes
- `publications`: selected papers with links
- `projects`: software, demos, datasets, notes
- `news`: recent timeline items

## Files

- `index.html`: page structure
- `styles.css`: visual design and responsive layout
- `script.js`: rendering, navigation, motion
- `site.config.js`: editable personal content
